Busstavlan

My Role: Programming | Concept Design

Busstavlan is a desktop application which is used to see when the next bus leaves from your nearest bus stop. You can search from point A to point B to get the next route, but if you travel a distinct route very often it can be tedious to type in the same route all the time. That's why you also can create your own customized route which you only have to click once to see when the next bus leaves.
